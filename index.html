HTML and CSS

  • HTML

    • This is the standard markup language for Web pages. It contains a series of elements that tell the browser how to display the content.

    History of HTML

    HTML was first concieved in 1989 where a Berners-Lee proposed the making of an internet based hypertext sytem. From then on events cascaded to the current version of HTML used ie HTML5.
    For more information on the history of HTML view the wikipedia page click here.
    HTML provides solutions to developers worldwide as many seek to have information on the internet.

    HTML uses elements to tell the browser how to display the information contained within the HTML document. Let's see more about these elements under the following heading.

    Element

    HTML elements are the defining items in a HTML document. They are characterised by opening and closing tags. The element itself is anything from the start to the end of the end tag. Most
    elements need the closing end tags but there are a few that stand alone for example the line break element. Such elements are called empty tags

    HTML Documents

    HTML documents must start with the declaration (!DOCTYPE HTML). This must appear once at the top of the document and its purpose is to help the browser better display the document. When you're online and you wonder how the page developers accomplished a certain feature, you can do so by using the control (Ctrl+Shift+I)

  • CSS

    • This stands for Cascading Sheets Styles. It has saved a lot of work since it can conrol the layout of multiple web pages at once.
    With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!
    NB: The word cascading means that a style applied to a parent element will also apply to all children elements within the parent.
